Chapter 1. Overview

The BeanShell Interpreter Bis plugin (BeanShellBis.jar) provides an interface to the instance of jEdit's inner BeanShell interpreter in the way that allows to render suitably written BeanShell scripts into jEdit plugins.

With this interface, a BeanShell script becomes the main body of jEdit's plugin, and methods of the BeanShell scripted object become dockable(s) or actions of this jEdit's plugin.

In the namespace of the BeanShellBis interpreter, the same packages are imported and the same variables are set, as for the inner jEdit's BeanShell script interpreter.

Chapter 2. Use

The use of this plugin is illustrated in the demo BeanShellObj plugin.

Objects of type BshMethod are accessed via getMethods() applied to the interpreter's namespace. Then, invoke() is applied to an object of type BshMethod.
